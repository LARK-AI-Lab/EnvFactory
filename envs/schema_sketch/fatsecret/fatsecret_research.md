# FatSecret Research Notes

## Data Source
- Official Docs: https://platform.fatsecret.com/platform-api
- Developer Portal: https://platform.fatsecret.com/
- Main Site: https://www.fatsecret.com/

## Authentication
- Type: OAuth 2.0 (3-legged)
- Also supports: OAuth 1.0a
- Free Basic tier available

## Core Endpoints
1. `foods.search` - Search foods by keyword
2. `food.get` - Get detailed food information
3. `foods.get_by_barcode` - Barcode lookup
4. `recipe.search` - Search recipes
5. `recipe.get` - Get recipe details
6. `food_entries.get_month` - Get user food diary

## Key Features
- 2.3+ million foods across 56 countries
- 24 languages supported
- 90%+ global barcode coverage
- 19,000+ curated international recipes
- AI-powered food image recognition
- Natural language processing for food input
- Allergen information (10 allergens)
- Dietary preference flags (vegan, vegetarian, etc.)

## Food Types
- Generic foods (ingredients)
- Composite generic foods (dishes)
- Branded foods (packaged products)
- Restaurant foods

## Premier Features
- Image recognition for nutrition analysis
- NLP text/voice input processing
- High-quality food images
- Detailed micronutrient data
- Professional SLA with support

## Notes
- Basic API free for testing
- Premier features require commercial license
- Flexible month-to-month licensing
- Used by major companies: Samsung, Fitbit, Nestlé, etc.
